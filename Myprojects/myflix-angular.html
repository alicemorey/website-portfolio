MyFlix Angular

Project Description

The MyFlix Angular Client is a front-end app built with Angular, made for users to browse a catalog of movies, directors, and genres. It has features like user login, profile management, and lets users save their favorite movies. This project shows my ability to create a responsive single-page app using Angular and includes clear documentation to help other developers understand and contribute.
