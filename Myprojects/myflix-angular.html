MyFlix Angular

MyFlix Angular is a client-side application built with Angular, and designed for user to access details of movies. The application includes user authentication and profile management page and allowing users to make a list of their favourite movies.

I developed an understanding of how to use Angular and build an app with it, including TypeScript. I also enhanced my UI skills and understanding by implementing Angular Material, which has build in tools and styles for Angular, to keep a consistency in the app user-ability.
